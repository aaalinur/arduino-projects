## 🛠️ General Tech Stack & Tools

- **Microcontrollers:** Arduino Uno, Arduino Nano (AVR architecture)
- **Programming Language:** C++ / Embedded C (Arduino Framework)
- **Concepts & Techniques:** Non-blocking timing (`millis()`, `micros()`), Multiplexing, Flash Memory optimization (`PROGMEM`), Direct pin control
- **IDE:** Arduino IDE 2.x / VS Code with PlatformIO

## 🚀 How to Navigate This Repo

1. Go to the specific project directory from the table above (e.g., [`matrix_countdown_music`](./matrix_countdown_music)).
2. Read the project's dedicated `README.md` to check pin configurations and schematic notes.
3. Open the `.ino` file in **Arduino IDE** and upload it to your board.
